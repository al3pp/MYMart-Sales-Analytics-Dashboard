# MY Mart - Sales Analytics Dashboard

An interactive Power BI dashboard designed to analyze sales performance, category trends, customer metrics, and logistical insights for MY Mart.

## Key Insights & Features
* **Overview:** Complete visibility into revenue growth trends, order volumes, and state-by-state sales distribution across Malaysia.
* **Customer Dynamics:** Tracking top high-value customers by total revenue contribution and purchasing behavior.
* **Product Optimization:** Visualized category breakdowns using clear, readable visual elements.
* **Logistics & Predictions:** Deep-dives into delivery efficiency and forecasting future sales trends.
